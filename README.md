# 🕹️ TapVerse Arcade
A high-performance, mobile-first web arcade built with Vanilla JS.

## 🚀 How to Play
1. Visit the live link (once deployed).
2. Use your **Mouse** or **Touchscreen** to control the games.
3. Earn coins to level up and unlock new content!

## 🛠️ Built With
- **HTML5 Canvas** for high-performance rendering.
- **Vanilla JavaScript** (No heavy frameworks).
- **CSS3 Variables** for dynamic theme switching.
- **LocalStorage** to save your progress forever.
